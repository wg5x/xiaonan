## 目录约定

每个项目使用一个英文目录名，项目内再统一放置：

```text
projects/<project-name>/
├── README.md
├── docs/
├── src/
├── assets/
└── scripts/
```

说明：

- `docs/`：需求文档、原始资料、报价、会议记录等。
- `src/`：后续项目源码。
- `assets/`：图片、原型、模板、截图等资源。
- `scripts/`：临时脚本、数据处理脚本、自动化工具。

后续新项目也按这个结构新增，例如：

```text
projects/example-project/
├── README.md
├── docs/
├── src/
├── assets/
└── scripts/
```
