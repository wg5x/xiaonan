## 文档内容概览

`AI标书写作平台-完整功能清单.docx` 主要包含：

- 项目概述：产品定位、商业模式、技术约束。
- 用户端架构：Web 形态、技术架构、前期准备事项。
- 功能清单：按阶段拆分基础架构、模板系统、智能图表、知识库、团队协作、运营后台等能力。
- 收费体系：会员费、Token 消耗费、支付接入。
- 范围边界：明确不做标讯商机搜索等功能。
- 技术方案：技术栈总览、Token 强制走自有中转的实现方案。

## 目录规划

- `docs/00-原始资料/`：客户原始资料、完整方案、外部输入文件。
- `docs/01-产品需求/`：已经整理过的 PRD、MVP 范围、功能定义。
- `docs/02-周期报价/`：开发周期、报价、成本、人力估算。
- `docs/03-新增需求/`：后续新来的需求，先按日期归档，评估后再并入 PRD。
